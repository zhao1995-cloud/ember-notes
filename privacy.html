焚火便签 隐私政策

最后更新日期:2026-05-28
一句话总结:本应用不收集、不上传、不分享任何 个人数据。所有任务、设置、主题偏好均存储在你本机,我们没有 任何服务器接收这些数据。

1. 我们收集什么信息

我们不收集任何信息。本应用是一款纯离线工具,不要求注册账号, 不要求登录,不要求任何个人身份信息(姓名、邮箱、手机、地理位置等)。

2. 你的数据存储在哪里

所有以下数据仅存储在你本机硬盘,具体路径:

%APPDATA%\焚火便签\

  • 任务列表(tasks.json)
  • 应用设置(settings.json,含主题、励志一句、自启时间等偏好)
  • 窗口位置(window-state.json)

我们(开发者)无法访问、读取或删除你机器上的这些文件。 卸载本应用即可彻底清除所有数据。

3. 我们不做的事

  • 不向任何服务器发送你的任务内容
  • 不收集你的使用行为(无埋点、无统计、无 Analytics)
  • 不向第三方分享、出售、出租你的任何信息
  • 不嵌入任何广告 SDK
  • 不读取你机器上其它应用的数据

4. Windows 系统级集成

如果你在设置里启用"每日定时自启",本应用会调用 Windows 自带的 schtasks 命令注册一个本地计划任务。这是 Windows 系统功能, 数据完全在你的机器内,不上传到任何地方。

5. 应用更新

本应用通过微软商店分发,更新由微软商店统一处理。本应用不主动 检查更新,也不会通过其它方式联网。

6. 儿童隐私

本应用不专门面向 13 岁以下儿童,但因为不收集任何信息, 儿童使用也不存在隐私问题。

7. 政策变更

如本政策有变更,我们会更新本页面顶部的"最后更新日期"。
